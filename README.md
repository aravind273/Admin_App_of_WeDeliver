# Admin_App_of_WeDeliver
This is  Admin app for WeDeliver application .This application is only for Admins in which they can add items,can add Laundry and rental services,can manage the orders placed by customers,can view registered customers details with his/her images,can analyse the users with the help of PieChart based on the orders placed by the customers and can answer the requests placed by customers and resolve their problems.<br>
